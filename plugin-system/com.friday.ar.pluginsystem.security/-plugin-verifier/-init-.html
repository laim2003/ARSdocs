PluginVerifier()

Use this class to verify any kind of possible PluginFile This class checks the following cases:

  • The file contains a manifest.
  • The manifest contains the required meta info
  • The manifest contains the required application info
  • The manifests meta info contains the following keys:applicationName, versionName
  • The manifests application info contains the following keys:class

Warning policy In case of missing and security relevant info, such as an identifier and/or name of the author of a plugin, a warning will be thrown. Plugins which are installed from the official Friday store are not allowed to have the required info missing and will be blocked from installing.
